# NiftyAITrader

A Python-based Nifty trading bot with paper trading support.

## Features

- Live Nifty data using Yahoo Finance
- EMA20 & EMA50 Strategy
- RSI Filter
- BUY / SELL Signals
- Paper Trading
- Trade Logging (CSV)
- Market Hours Filter
- Modular Project Structure

## Installation

```bash
pip install yfinance pandas ta
```

## Run

```bash
python main.py
```
